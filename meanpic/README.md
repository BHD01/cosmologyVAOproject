# meanpic — 平均图像生成

处理模拟程序 **21cmFAST-VAO** 生成的宇宙黎明时期 21 cm **二维多频率角功率谱** 图像，生成平均图像的程序。

## 文件说明

| 文件 | 说明 |
|------|------|
| `picture.py` | 主程序，生成平均图像 |
| `VAOpic.py` | VAO 信号图像处理 |
| `JLW.py` | LW 反馈相关图像处理 |
| `fixcsv.py` | CSV 数据修复 |
| `offsetpic.py` | 偏移校正图像处理 |
| `tmp.py` | 临时脚本 |
| `global/` | 全局数据文件（`.txt`） |

## 背景

- **21cmFAST-VAO**: 模拟宇宙黎明时期 21 cm 信号的程序
- **二维多频率角功率谱**: 描述 21 cm 信号在不同频率和角度上的分布
